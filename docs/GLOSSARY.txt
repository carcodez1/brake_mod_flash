SECTION 1 – TECHNICAL GLOSSARY
--------------------------------------------------------------------------------

Arduino Nano  
- A compact 5V microcontroller board based on the ATmega328P, operating at 16MHz.  
- Used to execute timing logic for the brake light pattern.  
- Must be powered by a regulated 5V input (not raw 12V).

ATmega328P  
- The microcontroller chip used in the Arduino Nano.  
- Contains 32KB flash memory, 1KB EEPROM, and multiple digital I/O pins.

Buck Converter (Step-Down Regulator)  
- A DC-DC voltage converter that safely steps down vehicle 12V to 5V.  
- Essential for powering the Arduino from a vehicle supply.  
- Must be rated for automotive input range (12–16V nominal, 24V surge).

Optocoupler (Opto-Isolator)  
- An electronic component that uses light to transmit signals between two circuits.  
- Protects the low-voltage Arduino from high-voltage vehicle signals.  
- Example: PC817, 4N35.

Voltage Divider  
- A two-resistor circuit that scales 12V down to ~5V.  
- Less protected than an optocoupler, but acceptable in fused, low-noise conditions.

MOSFET (Metal–Oxide–Semiconductor Field-Effect Transistor)  
- Acts as an electronic switch controlled by the Arduino to toggle the brake light.  
- Example: IRFZ44N, IRLZ44N (logic-level).

Flyback Diode  
- Installed across inductive loads (relays, solenoids) to prevent damaging voltage spikes.  
- Required when using a relay to switch the brake light.  
- Example: 1N4007.

TVS Diode (Transient Voltage Suppressor)  
- Clamps dangerous voltage spikes caused by alternator ripple or switching noise.  
- Protects the Arduino input and power lines from damage.  
- Example: SMAJ24A.

Flash Pattern  
- A sequence of ON and OFF durations configured via the emulator GUI.  
- Stored in a config file and embedded into the compiled firmware.

Firmware  
- The executable logic running on the Arduino.  
- Generated automatically using `render_config_to_ino.py`.  
- Output file: `BrakeFlasher.ino`.

GUI Emulator  
- Python-based graphical tool for designing and simulating flash patterns.  
- Allows real-time preview and config export.

EEPROM  
- Non-volatile memory on the Arduino for storing firmware metadata or user configuration.  
- Used to embed the firmware version and flash pattern signature.

Ground (GND)  
- The electrical return path for current.  
- All components (Arduino, buck converter, MOSFET, vehicle) must share a common ground.

FMVSS 108  
- Federal Motor Vehicle Safety Standard 108.  
- Governs U.S. regulations for brake and signal lights.  
- Flashing third brake lights are not legal in all states or jurisdictions.

SHA256 Checksum  
- A cryptographic hash used to verify firmware integrity.  
- Included in the metadata JSON and optionally printed in the CLI output.
